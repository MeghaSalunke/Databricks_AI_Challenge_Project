# Datasets Used

This folder contains publicly available datasets downloaded from Kaggle.
These datasets are used only for **training, testing, and validating** the system.

## Datasets Included
- **bronze_er_wait_time.csv**
  - Emergency room waiting time and triage data
- **bronze_admission_data.csv**
  - Hospital admission and discharge records
- **bronze_emergency_room_data.csv**
  - Daily emergency room patient inflow data

## Note
These datasets are anonymized and used only for educational and system design purposes.
In real-world deployment, hospital-owned data would replace these datasets.
